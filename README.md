# 🏥 Government Health Infrastructure Analysis — India

This project analyzes the distribution and accessibility of healthcare infrastructure across different states and districts in India.

## 📌 Objectives
- Analyze healthcare facilities distribution (PHCs, CHCs, Hospitals)
- Identify regional disparities
- Evaluate accessibility using population-to-facility ratios
- Detect underserved regions

## 📂 Dataset
- Rural Health Statistics (District-wise)

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## 📊 Key Insights
- Unequal distribution across states
- Some districts severely underserved
- High population-to-facility ratios in rural areas

## 🚀 How to Run
1. Clone the repo
2. Install dependencies:
  pip install -r requirements.txt
3. Open notebook: jupyter notebook
